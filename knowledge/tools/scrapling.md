# Scrapling

Undetectable, lightning-fast, and adaptive web scraping library for Python.

## Key Features

- **Adaptive Scraping**: Automatically adjusts to website structure changes using smart element tracking
- **Stealthy Fetching**: Bypasses anti-bot protections and detection systems
- **High Performance**: Up to 240x faster text extraction than BeautifulSoup4, 4.5x faster than AutoScraper
- **Developer-Friendly API**: Inspired by Scrapy and BeautifulSoup
- **Content-Based Selection**: Find elements by text content and locate similar elements
- **Multiple Fetchers**: Supports `Fetcher`, `StealthyFetcher`, and `PlayWrightFetcher` for various needs

## Use Cases

- Building resilient web scrapers that survive website structure changes
- High-speed data extraction and content parsing
- Circumventing anti-bot measures and rate limiting
- Automated content aggregation and monitoring

## Performance Benchmarks

- Text extraction: 240x faster than BeautifulSoup4
- Extraction by text: 4.5x faster than AutoScraper
